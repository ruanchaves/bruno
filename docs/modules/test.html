Functions

run_test (entry, lua, interpretador, buffer, buffer_mode, redirection)
Executa um teste.

Parameters:

  • entry Entrada na lista de testes.
  • lua Comando que invoca o interpretador de Lua. Por padrão, tem o valor "lua".
  • interpretador Comando que invoca o interpretador de BPL. Por padrão, tem o valor "interpretador.lua".
  • buffer Nome do arquivo temporário de buffer. Por padrão, tem o valor "buffer.txt".
  • buffer_mode Como a shell deve enviar sua saída ao arquivo de buffer. Por padrão, ele é sobrescrito ( ">" ).
  • redirection Para onde redirecionar mensagens de erro em Lua (stderr). Por padrão, redireciona para stdout ( 2>&1 ).

Returns:

    test_report Retorna a data do teste, a saída esperada e o resultado obtido.
record_results (results_filename, mode)
Executa todos os testes, gerenciando chamadas a run_test.

Parameters:

  • results_filename Arquivo onde os resultados devem ser salvos. Por padrão, "report.txt".
  • mode Qual modo de escrita deve ser adotado no arquivo de resultados. Por padrão, "a".

Returns:

    nil
