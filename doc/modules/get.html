get_value (command, run, verbose)
Retorna o valor correspondente a uma variável ou posição de array.

Parameters:

  • command Comando contendo variável.
  • run um objeto da classe Runner.
  • verbose Imprime mensagens do debugger.

Returns:

    Valor da variável ou posição de array.

Raises:

Retorna um erro para acesso a posição fora do alcance de um array.

get_result (num1, op, verbose)
Executa uma operação aritmética entre dois valores. Variáveis e funções precisam ser resolvidas em valores antes da chamada desta função.

Parameters:

  • num1 Primeiro valor.
  • op Operador. Operadores permitidos: adição ( + ), subtração ( - ), multiplicação ( * ) e divisão ( / ).
  • verbose Imprime as mensagens do debugger.
