create (function_list, function_list_tags)
Cria um objeto Runner.

Parameters:

  • function_list Lista de listas de comandos de funções.
  • function_list_tags Lista de listas que para cada comando em function_list, na mesma posição, apresenta a tag correspondente.

Encontra o índice de uma função em function_index a partir do seu nome.

Parameters:

  • name Nome da função.
execute (cmds, cmd_tags)
Executa uma lista de comandos associada a uma lista de tags.

Parameters:

  • cmds Comandos.
  • cmd_tags Lista que para cada comando em cmds, na mesma posição, apresenta a tag correspondente.
header (command)
Lê o header de uma função. Caso seja "main", inicializa a callstack. Caso contrário, ele é ignorado.

Parameters:

  • command Header de função.
vardef (command)
Lê uma definição de variável ou de array.

Parameters:

  • command Definição de variável ou array.
begin (command)
Lê um comando "begin", e em seguida, retorna um valor nulo.

Parameters:

  • command Comando "begin".

Returns:

    nil
end_ (command)
Lê um comando "end". Elimina o escopo da função atual, e adiciona um par chave : valor ao escopo da função que a chamou. A chave é o nome da função atual, e o valor é o seu valor de retorno.

Parameters:

  • command Comando "end".

Returns:

    nil
attr (command, verbose)
Lê uma atribuição de valor a variável, que pode ter um argumento ou uma operação aritmética entre dois argumentos.

Parameters:

  • command Comando com atribuição.
  • verbose Imprime mensagens do debugger.

Returns:

    nil

Raises:

Erro de acesso a índice fora do alcance do vetor.
